Pattern

The final page of a transactional service, which tells the user the outcome of the transaction and any next steps or actions.

This page is also used to measure how long it takes to complete a transaction using Google Analytics goal tracking.

The complete page should:

  • inform the user of the outcome
  • provide a reference number
  • advise of any notifications sent in relation to the transaction
  • tell the user what happens next
  • link to any related services
  • link to any survey for feedback

Example

Application submitted

Reference 12345

We have sent you an email with details of your application.

Next steps

It takes about 14 days to process your application.

We will contact you if we need any additional information from you.

Give us feedback about this service

Tell us what you thought about this service. (Takes about 1 minute)
