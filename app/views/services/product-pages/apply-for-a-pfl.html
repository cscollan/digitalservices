Apply for a Personal Functional Licence (PFL)

Live

Summary

Apply for a PFL is a service provided by the Gambling Commission to allow users to apply for a Personal Functional Licence. This will allow them to work within the casino industry.

User needs

If you plan to be involved in gaming or handling cash in relation to gambling at a casino you must hold a personal functional licence (PFL) before you start work.

You need a PFL if you are:

  • a dealer/croupier
  • a cashier
  • an inspector
  • a pit boss/gaming supervisor
  • security staff/monitoring surveillance related to gambling activities.

User stories

As a potential employee of a casino, I need to apply for and be granted a personal licence by the Gambling Commission, so that I am able to work.

This service meets the needs of the user when:

  • Application has been completed and submitted
  • Licence has been granted
  • The licencee is able to commence employement
